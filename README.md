# DotFiles

DotFiles on Arch Linux

## Setup ##
```
Copy config files to your $HOME directory
```

## Requirements ##

Make sure you have the following installed:

- i3 / i3wm
- termite
- compton
- feh
- zsh
- oh-my-zsh
- ttf-inconsolata
- ttf-roboto
- arc-icon-theme
- i3-gaps (AUR)
- i3lock-fancy-git (AUR)
- adapta-gtk-theme (AUR)
